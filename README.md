# DarijaAssist
DarijaAssist is a voice-first AI agent enabling illiterate users to access Moroccan government services (CNSS, AMO, CIN) via spoken Darija. Built with FastAPI, it runs a 7-step pipeline—Whisper ASR, NLLB-200 translation, FAISS-based RAG, GPT-4o reasoning, and Coqui TTS—to deliver grounded, audio-native responses without typing or reading.
